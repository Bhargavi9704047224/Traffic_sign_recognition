# 🚦 Traffic Sign Recognition using CNN

## 📌 Project Overview
This project is a deep learning-based **Traffic Sign Recognition System** that classifies traffic signs using a Convolutional Neural Network (CNN). The model is trained on a dataset of road signs and can predict the correct class of a sign from an input image.

---

## 🧠 Problem Statement
To automatically recognize and classify traffic signs from images, which is essential for autonomous driving systems and road safety applications.

## 📊 Dataset

The dataset used for this project is the **German Traffic Sign Recognition Benchmark (GTSRB)**.

📌 Dataset Link:  
https://www.kaggle.com/datasets/valentynsichkar/traffic-signs-dataset-in-yolo-format
---

## ⚙️ Tech Stack

---

## 📊 Dataset
- Dataset used: Traffic Sign Dataset (GTSRB or similar)
- Contains multiple classes of road signs
- Images resized to 30x30 pixels for model input

---

## 🏗️ Model Architecture
- Convolutional Neural Network (CNN)
- Conv2D + ReLU activation
- MaxPooling layers
- Dropout for regularization
- Dense + Softmax output layer

---

## 🚀 Workflow
1. Load dataset
2. Preprocess images (resize, normalize)
3. Encode labels
4. Train CNN model
5. Evaluate accuracy
6. Predict traffic signs on new images

---

## 📈 Results
- Achieved good accuracy on validation dataset
- Model successfully classifies multiple traffic sign categories
